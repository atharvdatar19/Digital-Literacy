# Digital Literacy Project

This repository contains the deliverables for the Digital Literacy Project, organized by tasks.

## Structure
- **report/**: Contains the full written project report.
- **task-1-presentation/**: Digital Literacy Infographic.
- **task-2-portfolio/**: Profile screenshots (GitHub, LinkedIn, Kaggle).
- **task-3-platforms/**: HackerRank challenge and Google Form screenshots.
- **task-4-email-etiquette/**: Professional email drafts and social media do's and don'ts.
- **task-5-cybercrime/**: UPI Fraud case study and online safety checklist.
